# AI Multi-Agent System

This folder holds configuration, prompts, and documentation for the multi-agent system running on Ollama.

## Agents List
* **RouterAgent**: Identifies intent (expense, income, note, report, reminder, unknown).
* **ExpenseAgent**: Extracts amount, category, description, payment method, transaction date.
* **NoteAgent**: Generates tags, titles, and summaries for notes.
* **ReportAgent**: Generates monthly reports and financial insights.
* **ReminderAgent**: Schedules and formats reminders.
* **FallbackAgent**: Requests clarification for ambiguous inputs.
