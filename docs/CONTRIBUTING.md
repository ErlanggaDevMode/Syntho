Contributing Guide

Terima kasih telah berkontribusi pada AI Notes & Expense Tracker.

Development Workflow
Fork repository.
Buat branch baru.
git checkout -b feature/nama-fitur
Lakukan perubahan.
Pastikan seluruh pengujian berhasil.
Commit menggunakan format konvensional.
Push branch.
Buat Pull Request.
Branch Naming Convention
feature/<nama-fitur>
fix/<nama-bug>
docs/<nama-dokumen>
refactor/<nama-refactor>
chore/<nama-task>

Contoh:

feature/telegram-expense-parser
fix/auth-refresh-token
docs/update-prd
Commit Convention

Gunakan Conventional Commits.

feat: add telegram expense parser
fix: resolve jwt expiration issue
docs: update deployment guide
refactor: simplify ai routing service
test: add transaction integration tests
chore: update dependencies
Pull Request Checklist

Kode sudah diuji

Dokumentasi diperbarui

Tidak ada kredensial sensitif

Linter berhasil

Seluruh test berhasil

PR memiliki deskripsi yang jelas
